# Smart Motorbike Parking Management System (CNPM)

A modern, SaaS-ready parking management system built with Next.js 16 and React 19.

## 🚀 Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

1. Navigate to the frontend directory:
   ```bash
   cd frontend
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

### Running the App

Start the development server:
```bash
npm run dev
```

The application will be available at [http://localhost:3000](http://localhost:3000).

## 🛠 Tech Stack

- **Frontend**: Next.js 16 (App Router), React 19
- **Styling**: Tailwind CSS 4, Lucide React (Icons)
- **State Management**: Zustand
- **Form Handling**: React Hook Form, Zod
- **Data Fetching**: TanStack Query (React Query)
- **UI Components**: Shadcn UI, Base UI, Radix UI
- **Visuals**: Recharts (Analytics), Sonner (Toasts)

## 📁 Project Structure

- `frontend/`: Next.js application source code.
- `docs/`: Project documentation and assets.
